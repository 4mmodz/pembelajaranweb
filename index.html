Pembelajaran Dasar Pembuatan Website

HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar dari sebuah halaman website. Bisa dibilang HTML itu kerangka atau fondasi utama sebuah web.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello World</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf.</p>
  </body>
</html>

CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan website seperti warna, font, dan layout.

body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

h1 {
  color: navy;
  text-align: center;
}

JavaScript

JavaScript digunakan untuk menambahkan interaktivitas pada website.

<button onclick="alert('Halo!')">Klik Aku</button>

<script>
  function halo() {
    alert("Selamat datang di pembelajaran web!");
  }
  halo();
</script>
